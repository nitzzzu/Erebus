# Songwriting & AI Music

Use this skill for songwriting assistance and AI music generation.

## When to Use

- User wants to write song lyrics
- User wants to generate music with Suno or Udio
- User needs help with melody or chord suggestions
- User wants to adapt or parody an existing song

## Song Structure

```
[Intro]
[Verse 1]
[Pre-Chorus]
[Chorus]
[Verse 2]
[Pre-Chorus]
[Chorus]
[Bridge]
[Chorus]
[Outro]
```

## Lyric Techniques

- **Rhyme schemes**: AABB, ABAB, ABCB
- **Internal rhyme**: Rhymes within the same line
- **Assonance**: Repeated vowel sounds
- **Alliteration**: Repeated consonant sounds
- **Imagery**: Concrete, sensory details
- **Metaphor**: Abstract concepts through comparison

## AI Music Prompts (Suno/Udio Format)

```
[Genre: indie folk, acoustic]
[Mood: wistful, nostalgic]
[Tempo: 110 BPM]
[Key: G major]
[Instruments: acoustic guitar, harmonica, soft drums]

[Verse 1]
Walking down the old dirt road...
```

## Process

1. **Brief**: Understand the mood, genre, and theme
2. **Structure**: Choose song structure and rhyme scheme
3. **Write**: Draft lyrics with attention to meter and flow
4. **Format**: Add metatags for AI generation if requested
5. **Refine**: Polish lyrics for rhythm and impact
